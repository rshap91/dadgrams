# dadgrams


#### Install nvm

https://github.com/nvm-sh/nvm

`curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash`

#### Install Node

`nvm install node`


#### Install Parcel

`npm install parcel`


#### Run


```
cd app

npm start
```

I had some issues. I deleted dist and node_module and then ran it again and it seemed to work.
